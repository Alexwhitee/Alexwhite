## 术数概览

### 一、“术数”是什么

“术数”是一种占卜方法，用于预测和辅助理解易理。

- **基础与特征：** 特征是以“数”行方术；基础是阴阳五行、天干地支、河图洛书等。
- **“术”与“数”：** “术”指方术；“数”指气数、数理（阴阳五行、太极八卦的数理）。

### 二、术数博览

#### 1、东方玄学体系

**命理（通过出生时间对人一生命运进行预测）**

- **四柱/八字**
  
    - **核心定义：** 一个人出生时的干支历日期，年、月、日、时共四柱干支，合共八个字。
      
    - **推断机制：** 以十神及宫位来推断人的命运。
      
    - **起卦方式：** 以**出生年月日时**排定八字命盘。
    
- **紫微斗数**
  
    - **系统：** 古代印度占星学与中国阴阳五行的结合。
      
    - **起卦方式：** 以**出生年月日时**定盘，排布十二宫和星曜。
      

**卜卦（依托伏羲八卦和文王六十四卦进行占算推演的卜术）**

- **六爻（金钱卦）**
  
    - **定义：** 由大衍筮法发展而来，因“以钱代蓍”，也称“金钱卦”。
      
    - **推断：** 配以卦爻纳甲、动变，通过当日干支、六亲及六神推断事物发展。
      
    - **起卦方式：** **摇晃三枚铜钱六次**成卦，推荐使用模拟起卦。
    
- **梅花易数（梅花心易）**
  
    - **创始人：** 北宋**邵雍**（邵康节先生）所创，由《皇极经世书》简易而来。
      
    - **特点：** 卦式简单，即排即断，灵活多变。占算很靠人的“灵感”。
      
    - **起卦方式：** 随意**报数、时间、字数、物件数量**等外应起卦。
    
- **奇门遁甲**
  
    - **本质：** 一门高等的**天文物理学**。
      
    - **构成：** 由“三奇”（乙、丙、丁）、“八门”（开、休、生、伤、杜、景、惊、死）、“遁甲”组成。
      
    - **推断机制：** 将一切成败归纳为五大因素：天时、地利、人和、神助、格局组合。
      
    - **起卦方式：** 推荐使用当下时间起卦。
    
- **六壬（大六壬）**
  
    - **定义：** 中国古代宫廷占术，以日、月、星、辰变化为准则。
      
    - **推断：** 推导包括四课和三传。
      
    - **起卦方式：** 推荐使用当下时间起卦。月将选取推荐默认的中气换将。
    
- **小六壬**
  
    - **特点：** 一种快速、简便的掐指速算占卜法。
      
    - **推断：** 以六个掌上宫位（大安、留连、速喜、赤口、小吉、空亡）进行推算。
      
    - **起卦方式：** 推荐使用当下日时起卦
      

#### 2、西方玄学体系

- **占星（Astrology）**
  
    - **定义：** 一种根据天体运行和相对位置来判断或预测人间事务和命运的术数。
      
    - **推断：** 主要以**行星、星座、宫位、相位**四大要素进行推断。
      
    - **起卦方式：** 以**出生时间、地点**绘制**本命盘，或以**问事时间**绘制**卜卦盘。
    
- **塔罗（Tarot）**
  
    - **定义：** 一种使用塔罗牌（通常78张）进行占卜预测的工具。
      
    - **推断：** 通过牌面图案、象征意义、牌阵位置及占卜师的直觉进行解读。
      
    - **起卦方式：推荐用经典韦特塔罗的时间流牌阵。
    
- **占星骰子（Astrology Dice）**
  
    - **定义：** 一种结合了占星学要素的快速占卜工具，通常由三颗骰子组成。
      
    - **推断：** 三颗骰子分别代表**行星**（行动者）、**星座**（特质）和**宫位**（领域）。

### 三、东玄的底层逻辑（核心基础）

中国的玄学术数体系无论是风水、占卜、命理，基本都是在以下基础之上进行构建的：

#### 1、核心要素：阴阳、五行、干支

| 核心要素     | 角色定位                                                     |
| ------------ | ------------------------------------------------------------ |
| **阴阳**     | **吉凶的依据**。性质相反相成、互相对立又互相依赖的两种力量。 |
| **五行**     | **阴阳的具体状态、作用变化**。金、木、水、火、土。           |
| **干支**     | 阴阳五行结合的具体表示，是方位、时空与周期，**信息含量最高**。天干（10个）和地支（12个）。 |
| **其他基础** | 先后天八卦、河图洛书、日月五星。                             |

👉 **底层逻辑：** 这些符号通过不同的排列组合，设定特殊的计算方法便可以得到一种**数学模型（术数）**，再借用**类比**人的实际需求进行转换。

#### 2、阴阳的基本特性

- **定义：** **一阴一阳之谓道。**
  
- **本义体现：** 风水学说“**山南水北为阳**”（指代容易接受到阳光的位置）。
  
- **特点：** **对立统一**，可以**相互转换**（如阳极而衰）。
  
- **应用核心：** **清浊上下**，运用**类比**延伸。
  

#### 3、五行生克与基础知识

- **相生：** 木生火 → 火生土 → 土生金 → 金生水 → 水生木。
  
- **相克：** 金克木 → 木克土 → 土克水 → 水克火 → 火克金。
  

| 五行   | 方位（空间） | 时间         | 阴阳属性         | 五行本义（《洪范》） |
| ------ | ------------ | ------------ | ---------------- | -------------------- |
| **木** | 东方，青龙   | 春天         | 阳（上升，积极） | 曲直（塑造形状）     |
| **火** | 南方，朱雀   | 夏天         | 阳（上升，积极） | 炎上（上燃）         |
| **土** | 中央位置     | 四季最后18天 | -                | 稼穑（滋养作物）     |
| **金** | 西方，白虎   | 秋天         | 阴（下降，消极） | 从革（熔炼器物）     |
| **水** | 北方，玄武   | 冬天         | 阴（下降，消极） | 润下（下落滋润）     |

#### 4、先后天八卦

**（1）先天八卦（伏羲八卦）**

- **推演顺序：** 太极 → 两仪 → 四象 → **八卦**。
  
- **画卦顺序：** 自下而上，三爻象征着“天人地”。
  
- **口诀：** 描述天地万物的对立统一关系：**天地定位、山泽通气、雷风相薄、水火不相射。**
  

| 卦名   | 符号 | 自然象征 | 朱熹《八卦取象歌》 |
| ------ | ---- | -------- | ------------------ |
| **乾** | ☰    | 天       | **乾三连**         |
| **坤** | ☷    | 地       | **坤六断**         |
| **坎** | ☵    | 水       | **坎中满**         |
| **离** | ☲    | 火       | **离中虚**         |

**（2）后天八卦（洛书九宫）**

- **洛书：** 一三阶幻方，纵横交错相加皆为 **15**。
  
- **方位（洛书九宫）：** **戴九履一，左三右七，二四为肩，六八为足，五居中。**
  

| 卦名（洛书数） | 五行 | 空间（方位） |
| -------------- | ---- | ------------ |
| 离 (9)         | 火   | 正南         |
| 坎 (1)         | 水   | 正北         |
| 震 (3)         | 木   | 正东         |
| 兑 (7)         | 金   | 正西         |

#### 5、天干地支

- **组成：** 天干（10位） + 地支（12位） = **六十甲子**（用来表达年月日时）。
  

| 要素     | 数量 | 阴阳区分                                               | 五行分配（阳/阴）                                     |
| -------- | ---- | ------------------------------------------------------ | ----------------------------------------------------- |
| **天干** | 10   | 阳：甲、丙、戊、庚、壬；阴：乙、丁、己、辛、癸         | 木：甲/乙；火：丙/丁；土：戊/己；金：庚/辛；水：壬/癸 |
| **地支** | 12   | 阳：子、寅、辰、午、申、戌；阴：丑、卯、巳、未、酉、亥 |                                                       |

- **天干地支关系**
  
    - **天干五合：** 天干之间发生“化学反应”，如：**甲己合化土**（中正之合）。含义涉及贵人帮扶、情感、体制等。
      
    - **地支三合：** 地支之间间隔 4 个地支，形成一个**等边三角形，最为稳固**。本质是天文历法中的**四分法**。局如：**寅午戌合火局**、**申子辰合水局**、**巳酉丑合金局**、**亥卯未合木局**（**没有土局**）。
      
    - **地支六合：** 本质是**日月行次之合**，暗示与天文**十二黄道宫**的关系。有**克合、破合、生合、刑合**等不同属性。
